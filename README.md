# 📝 To-Do List Web App

A simple and responsive To-Do List web application built using **HTML, CSS, and JavaScript** that helps users manage daily tasks efficiently.

## 🚀 Features

- ✅ Add new tasks  
- ✏️ Mark tasks as completed  
- 🗑️ Delete tasks  
- 💾 Data persistence using localStorage  
- 📱 Clean and responsive UI  

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)  
- Browser localStorage API  

## 📌 How It Works

Tasks entered by the user are stored in the browser's **localStorage**, allowing them to persist even after refreshing or reopening the page. Users can toggle completion status and remove tasks easily.

## 🎯 Learning Outcomes

This project demonstrates:

- DOM manipulation  
- Event handling  
- CRUD operations  
- Basic state management  
- Local storage usage
